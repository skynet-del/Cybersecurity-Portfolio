This simulation focused on supporting a client during a cybersecurity breach investigation. The work involved reviewing web activity logs, identifying suspicious user activity, and answering investigative questions to support breach analysis.

## What I Did

I reviewed simulated web activity logs to identify suspicious user behavior and support a cybersecurity breach investigation. I analyzed activity patterns, answered investigation-focused questions, and practiced connecting log evidence to potential security concerns.

This simulation helped me build comfort with reviewing technical information, identifying indicators of suspicious activity, and documenting observations in a way that supports a larger investigation.

## Key Takeaways

This simulation strengthened my understanding of how log analysis supports cybersecurity investigations. It also reinforced the importance of careful review, attention to detail, and clear documentation when investigating suspicious activity.
