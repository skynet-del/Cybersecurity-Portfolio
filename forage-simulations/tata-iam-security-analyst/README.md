## What I Did

I reviewed a simulated cybersecurity consulting scenario focused on identity and access management. I practiced applying IAM concepts to help support secure access decisions and business-aligned cybersecurity recommendations.

I also developed documentation and presentation-style communication to explain cybersecurity concepts in a clear, professional way. This helped strengthen my ability to translate technical security ideas into recommendations that non-technical stakeholders can understand.

## Key Takeaways

This simulation helped me understand how IAM supports security, compliance, and business operations. It also reinforced the importance of access control, least privilege, documentation, and clear communication when working with stakeholders.
