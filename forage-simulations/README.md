# Forage Cybersecurity & Technology Simulation Experience

This section highlights cybersecurity, IT support, and technology-focused simulation work completed through Forage. These simulations allowed me to practice realistic tasks related to incident response, identity and access management, risk assessment, phishing awareness, fraud detection, Splunk dashboarding, service desk operations, log analysis, vulnerability research, and client communication.

The purpose of this section is to show how I approach cybersecurity and IT scenarios, document findings, communicate recommendations, and connect technical concepts to real-world business needs.

## Featured Simulation Areas

### Commonwealth Bank Introduction to Cybersecurity

Focused on fraud detection, Splunk dashboarding, incident response, password security awareness, penetration testing fundamentals, and web application security recommendations.

### Tata Cybersecurity Security Analyst

Focused on identity and access management, access control, cybersecurity best practices, business alignment, documentation, and technical communication.

### AIG Shields Up Cybersecurity

Focused on vulnerability research, CISA-related threat analysis, remediation communication, and ethical hacking concepts using Python.

### Datacom Service Desk

Focused on ITIL-aligned incident logging, ticket prioritization, escalation, user communication, troubleshooting, and service desk operations.

### Datacom Cybersecurity

Focused on cyberattack investigation, report writing, risk assessment, and recommendations to improve a client’s cybersecurity posture.

### Mastercard Cybersecurity

Focused on phishing threat identification, security awareness, business risk analysis, and training recommendations.

### Deloitte Australia Cyber

Focused on web activity log review, suspicious user activity analysis, and cybersecurity breach investigation support.

### Commonwealth Bank Tech Explorer

Focused on technology roles, stakeholder requirements, access groups, cybersecurity, software engineering, technology analysis, and data-focused decision making.

### Third Bridge Client Service Delivery

Focused on industry research, value chain analysis, expert profile development, client insight delivery, and persuasive presentation skills.

## Note

These simulations are presented as learning-based case studies. Each section summarizes the scenario, skills practiced, tools and concepts used, and key takeaways from the experience.
