# Commonwealth Bank Introduction to Cybersecurity Simulation

## Overview

This simulation involved working through tasks aligned with the role of a cybersecurity generalist supporting Commonwealth Bank’s cybersecurity team. The experience focused on fraud detection, incident response, security awareness, data visualization, and web application security concepts.

## Skills Demonstrated

* Fraud detection and prevention
* Splunk dashboarding
* Cybersecurity incident response
* Security awareness communication
* Password security best practices
* Web application security review
* Penetration testing fundamentals
* Technical documentation

## Tools & Concepts

* Splunk
* Historical customer data analysis
* Incident response process
* Password management guidance
* Web application vulnerability assessment
* Cybersecurity recommendations
* Security awareness design

## What I Did

I practiced using Splunk dashboards to identify patterns and insights in historical customer data that could support fraud detection. I also worked through cybersecurity incident response steps, including notifying relevant teams, collecting information, supporting containment, stopping attacks, and assisting with recovery.

The simulation also included creating security awareness material focused on secure password management and reviewing web application security concerns. I practiced identifying vulnerabilities and providing recommendations to strengthen cybersecurity defenses.

## Key Takeaways

This simulation helped me better understand how cybersecurity teams support fraud detection, incident response, security awareness, and application security. It also strengthened my ability to connect technical analysis with business risk and communicate security recommendations clearly.
