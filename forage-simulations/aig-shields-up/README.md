# AIG Shields Up Cybersecurity Simulation

## Overview

This simulation focused on cybersecurity threat analysis and vulnerability response as part of a Cyber Defense Unit scenario. The work involved reviewing reported vulnerabilities, researching security guidance, communicating remediation steps, and practicing technical problem-solving in an ethical hacking context.

## What I Did

I researched reported cybersecurity vulnerabilities and reviewed threat information to better understand potential security risks. I practiced identifying key remediation steps and drafted a clear communication to guide teams through vulnerability response.

The simulation also included a Python-based ethical hacking exercise where I practiced using scripting concepts to support decryption key recovery in a controlled scenario. This helped me connect technical problem-solving with real-world cybersecurity response work.

## Key Takeaways

This simulation helped me understand how cybersecurity teams review vulnerability information, communicate remediation steps, and support defense operations. It also reinforced the importance of staying current with trusted cybersecurity advisories and explaining technical risks clearly to affected teams.
