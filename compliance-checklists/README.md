## Controls and Compliance Checklist – Botium Toys

This document assesses the implementation status of key security controls and compliance best practices for Botium Toys. It includes evaluations related to PCI DSS, GDPR, and SOC frameworks.

It also contains a set of recommendations developed based on a mock audit, highlighting areas such as:
- Enforcement of least privilege
- Disaster recovery planning
- IDS/IPS deployment
- Secure data encryption
- Full PCI DSS alignment
